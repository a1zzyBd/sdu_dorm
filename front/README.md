# 🏠 SDU Dorm - Система управления общежитием

Веб-приложение для управления студенческим общежитием SDU, созданное с использованием **React**, **TypeScript**, **Vite** и **Tailwind CSS**.

## ⚙️ Требования

Перед началом работы убедитесь, что на вашем компьютере установлены:

- **Node.js** версии 18 или выше ([Скачать здесь](https://nodejs.org/))
- **npm** (устанавливается автоматически с Node.js) или **yarn**
- **Git** ([Скачать здесь](https://git-scm.com/))

### Проверка установки

Откройте терминал и выполните команды:

```bash
node --version   # Должна вывести версию Node.js (например, v20.x.x)
npm --version    # Должна вывести версию npm (например, 10.x.x)
git --version    # Должна вывести версию Git (например, 2.x.x)
```

Если какая-то из команд не работает, установите соответствующий инструмент по ссылкам выше.

## 🚀 Установка и запуск

### Шаг 1: Клонирование репозитория

Откройте терминал и выполните команду:

```bash
git clone https://github.com/Nurdaulet-no/sdu-dorm.git
```

Перейдите в папку проекта:

```bash
cd sdu-dorm/sdudorm-frontend
```

### Шаг 2: Установка зависимостей

Установите все необходимые пакеты:

```bash
npm install
```

**Примечание:** Процесс может занять несколько минут. Подождите, пока установятся все зависимости.

### Шаг 3: Запуск проекта

Запустите проект в режиме разработки:

```bash
npm run dev
```

После успешного запуска вы увидите сообщение примерно такого вида:

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

### Шаг 4: Открытие в браузере

Откройте браузер и перейдите по адресу:

```
http://localhost:5173/
```

**Готово! 🎉** Приложение запущено и готово к использованию.

## ✨ Функционал приложения

### Главная страница
- **Hero секция** с приветствием
- **О нас** - информация об общежитии
- **Галерея** - 3D карусель с фотографиями
- **Услуги** - список удобств в комнатах
- **Контакты** - платежные реквизиты и контактная информация

### Студенческий кабинет (после входа)
- **Home Page** - информация о студенте
- **News Page** - новости общежития (турниры, события)
- **Room Tracking** - интерактивная карта комнат (Блок A и B)
  - Просмотр доступности комнат
  - Выбор комнаты
  - Изменение статуса (занято/на обслуживании)
- **Rules and Regulations** - правила общежития и контракт
- **Online Services** - чат с персоналом общежития
  - Общение с координатором
  - Техническая поддержка
  - Помощь от ассистентов
- **Settings** - настройки интерфейса
  - Выбор языка (English/Kazakh/Russian)
  - Смена пароля
- **My Profile** - профиль студента
  - Просмотр личных данных
  - Редактирование информации

### Аутентификация
- Страница входа (Login)
- Страница регистрации (Sign Up)

## ❗ Устранение проблем

### Проблема: Порт 5173 уже используется

Если порт занят, Vite автоматически предложит другой порт. Следуйте инструкциям в терминале.

Или вручную укажите другой порт:

```bash
npm run dev -- --port 3000
```

### Проблема: Ошибки при установке зависимостей

Попробуйте очистить кэш npm и переустановить:

```bash
rm -rf node_modules package-lock.json
npm install
```

На Windows используйте:

```bash
rmdir /s /q node_modules
del package-lock.json
npm install
```

### Проблема: Белый экран или ошибки в консоли браузера

1. Откройте инструменты разработчика (F12)
2. Проверьте вкладку Console на наличие ошибок
3. Убедитесь, что все файлы в папке `public/` на месте
4. Перезапустите сервер разработки:

```bash
# Нажмите Ctrl+C в терминале, затем
npm run dev
```

### Проблема: Изменения не отображаются

1. Обновите страницу с очисткой кэша: `Ctrl+Shift+R` (Windows/Linux) или `Cmd+Shift+R` (Mac)
2. Убедитесь, что сервер разработки запущен
3. Проверьте, что вы редактируете правильный файл
